## New at this version

| Area | Change |
|------|--------|
| `figma-ds-modernization.md` Protocol 17 | New: Documentation Panel Consistency Audit — `scanPanelConsistency()`, `fixPanelConsistency()`, `scanPanelTextHierarchy()` |
| `scanPanelConsistency()` | Scans all doc panels on currentPage for fill drift (not `background/subtle`), radius drift (not `radius/lg`), text token drift |
| `fixPanelConsistency()` | Auto-corrects drifted fills and radii to canonical tokens; rebinds via `setBoundVariableForPaint` and `setBoundVariable` |
| `scanPanelTextHierarchy()` | Detects text nodes not following the 3-level hierarchy: `accent/primary` (headers) → `text/primary` (labels) → `text/tertiary` (descriptions) |
| Phase 0b `unscopedPrimitives` | New metric: counts `Color/Primitives` variables with `scopes.length > 0` (i.e., still visible in design panel pickers) |
| Protocol 8 Tier 4 | Tier 4 — Mature now requires `unscopedPrimitives = 0` in addition to `overallScore ≥ 85%` |
| SKILL.md Phase 0b reading questions | `unscopedPrimitives` question added: "Are primitive variables locked from the design panel?" |
| SKILL.md Protocol 7 Critical API Rules | Font loading pre-check rule added: `listAvailableFontsAsync()` must be called before any `setTextStyleIdAsync`; throws if font family unavailable (e.g. custom fonts at OFFICIAL tier) |
| SKILL.md Task Router | Protocol 17 row added: "audit doc panels", "panels look inconsistent", "documentation consistency" → `figma-ds-modernization.md` Protocol 17 |
| SKILL.md Quality Gate | Protocol 17 consistency check item added |
| Shared Helpers | `resolveAlias(varId)` added — walks `VARIABLE_ALIAS` chains until reaching a concrete `{r,g,b,a}` value; required for M2 color-matching |
